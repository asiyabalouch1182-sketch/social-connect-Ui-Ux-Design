# 🌐 Social Connect App – UI/UX Design

This repository contains my UI/UX design work for a **Social Connect App**, created as part of my internship project at Internee.pk.

## 🚀 Project Overview
The goal of this project was to design a modern, minimal, and user-friendly social media platform that enhances user interaction and engagement.

## 🎯 Features
- Clean and modern UI design
- User-friendly navigation
- Responsive layout (mobile & desktop)
- Profile customization
- Post feed with grid layout
- Dark/Light mode support

## 🖌️ Design Highlights
- Focus on simplicity and usability
- Consistent color system and typography
- Improved user flow and accessibility
- Interactive and engaging layout

## 🛠️ Tools Used
- Figma
  
## 📂 Project Structure

- Components & Style Guide
